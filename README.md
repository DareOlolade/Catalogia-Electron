# catalogia
A local-first PDF book catalog for your desktop.

## What it does
2-3 sentences. Real specifics.
- Import PDFs one at a time, in bulk, or entire folders
- Auto-extracts covers and metadata (title, author)
- Search across your library
- Categorize and track reading progress
- Everything stays on your machine — no cloud, no account

## Features
- Bulk import from folders (recursive)
- PDF metadata extraction
- Cover thumbnail generation
- Search by title or author
- Read status tracking (Not Read / Reading / Completed)
- Category filtering
- Dark theme

## Tech Stack
- Electron
- React
- Node.js
- Libraries: pdf-lib, pdf-to-img
